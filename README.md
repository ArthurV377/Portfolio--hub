# Portfolio Hub

Este projeto consiste em um portfólio pessoal desenvolvido com HTML, CSS e JavaScript, com o objetivo de apresentar projetos acadêmicos, certificados e interesses na área de tecnologia.

## 🚀 Objetivo
Criar um site estático hospedado no GitHub Pages para demonstrar habilidades em desenvolvimento front-end e organização de projetos utilizando controle de versão com Git.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- Git e GitHub
- GitHub Pages

## 📁 Estrutura do projeto
- `/assets` → imagens do site (certificados, basquete, etc.)
- `/css` → estilização do site
- `/js` → scripts JavaScript
- `index.html` → página principal
